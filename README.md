# config
`kp` expects you to have the following environment variables available:   
- `KP_URL` - the k8s dashboard URL.   
- `KP_TOKEN` - authorization bearer token.  
- `KP_ENV` - default env.
  
# usage
run `kp` or `kp --help` to see available commands and options.
